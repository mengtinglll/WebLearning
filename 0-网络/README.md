### OSI 七层协议
 - 应用层
   - 为应用提供通信服务HTTP HTTPS Websocket
 - 表示层
   - 定义数据格式以及加密
 - 会话层
   - 定义了如何开始、控制、结束一个会话，包括对多个双向消息的控制和管理。
 - 传输层
   - 选择差错恢复协议还是无差错恢复协议
   - TCP、UDP
 - 网络层
   - 端到端包传输。
   - 路由选择、包分解成更小的包
 - 数据链路层
   - 定义单个链路上如何传输数据
 - 物理层
   - 传输介质相关
